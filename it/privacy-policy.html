Usiamo le API di Google Docs e Google Drive per leggere un documento condiviso ed elecare metadati dei file contenuti in una cartella condivisa (URL di immagini e video), così che chiunque acceda a questo sito possa leggere frasi estratte dal documento e vedere foto prese dalla cartella.

This website does not accesses, uses, stores, or shares any user data.

We own both the document and the folder, so no user data are needed.
